
# JasaWebsiteKu Studio

Proyek ini dikembangkan menggunakan Firebase Studio.

## Cara Menghubungkan ke GitHub

Untuk mendorong (push) kode Anda ke repositori GitHub yang baru:

1. Buka terminal di direktori proyek Anda.
2. Jalankan perintah berikut:

```bash
git init
git remote add origin https://github.com/jackcarebian/studio.git
git add .
git commit -m "Initial commit dari Firebase Studio"
git branch -M main
git push -u origin main
```

## Teknologi yang Digunakan
- **Next.js 15** (App Router)
- **Genkit** (AI Perkiraan Biaya)
- **Firebase** (Firestore & Auth)
- **Tailwind CSS & Shadcn UI**
