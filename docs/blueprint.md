# **App Name**: JasaWebsiteKu

## Core Features:

- Halaman Beranda (Landing Page): Menampilkan informasi layanan pembuatan website dan aplikasi, keunggulan, serta testimoni pelanggan.
- Formulir Kontak: Memungkinkan pengunjung mengirimkan pesan atau permintaan penawaran melalui formulir yang terhubung ke email: promone.info@gmail.com.
- Detail Layanan: Halaman detail untuk masing-masing layanan (Website, Aplikasi Custom) dengan penjelasan fitur, teknologi yang digunakan (React, Next.js, TypeScript, Tailwind CSS, Shadcn/UI, PWA, Firestore), dan contoh hasil.
- Sistem Otentikasi Multi-Peran: Implementasi sistem otentikasi dengan peran berbeda (admin, user) menggunakan Firebase Authentication. Fitur termasuk registrasi, login, lupa password, dan proteksi rute.
- Estimasi Biaya Otomatis dengan Gemini AI: Menggunakan Gemini AI tool untuk menganalisis kebutuhan pengguna dan memberikan estimasi biaya pembuatan website/aplikasi dalam Rupiah. AI menjelaskan fitur yang dibutuhkan dan rincian biaya.
- Integrasi Whatsapp: Tombol atau tautan langsung ke nomor Whatsapp +62895803501000 untuk konsultasi cepat.
- Manajemen Data Firestore: Implementasi fitur menyimpan dan mengambil data dari database NoSQL yang skalabel dan real-time Firestore.

## Style Guidelines:

- Primary color: Light blue (#199EBD) to convey freshness and innovation.
- Background color: Off-white (#FAFAFA) for a clean and modern feel.
- Accent color: Pale green (#1CD3A2) to highlight key sections and calls to action.
- Body font: 'PT Sans', a humanist sans-serif suitable for body text.
- Headline font: 'Space Grotesk' sans-serif that gives a computerized and techy vibe.
- Gunakan ikon yang modern dan sesuai dengan tema teknologi untuk memperjelas fitur dan layanan.
- Tata letak responsif yang menyesuaikan dengan berbagai ukuran layar (desktop, tablet, mobile). Pastikan navigasi mudah dan informasi tersusun dengan baik.
- Animasi halus saat transisi antar halaman atau saat elemen muncul untuk memberikan kesan interaktif dan modern.
